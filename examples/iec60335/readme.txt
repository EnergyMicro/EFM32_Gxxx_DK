IEC60335 Class B selftest library example.

This example project shows how to use the IEC60335 Class B selftest library.
Use a debugger and step through the code in main() to study the behaviour
of the selftest library.
Refer to the documentation within the library (in the "reptile" folder) for
a more detailed description of the functionality provided by the selftest library.

Board:  Energy Micro EFM32-Gxxx-DK Development Kit
Device: EFM32G290F128 and EFM32G890F128
